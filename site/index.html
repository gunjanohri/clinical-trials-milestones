ClinicalTrials.gov completion watch

Current Phase 3 completions in 2026

This dashboard tracks the 949 current industry-sponsored interventional studies in the exported Phase 3 set whose primary completion date falls in 2026. The month view is built directly from the ClinicalTrials.gov export in phase3_current_industry_contains_phase3.csv.

Scope: current records whose phase list contains PHASE3, with statuses limited to NOT_YET_RECRUITING, RECRUITING, ENROLLING_BY_INVITATION, ACTIVE_NOT_RECRUITING, SUSPENDED. Source data timestamp: 2026-05-15T09:00:05.
